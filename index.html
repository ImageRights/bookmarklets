ImageRights Bookmarklets & Tools

Drag a bookmarklet link below into your bookmarks bar to use it.

Bookmarklets

  • Find Image: Automatically scroll through an image set to find an image. To use: Search for an image by name. Click the link to open the image set. Click the bookmarklet. It will ask you to enter a time in milliseconds to wait between page loads, defaulting to 500 ms. Enter what your computer can handle. If you enter too short a time, pages won't have enough time to load before the script skips to the next page. The target image will be highlighted once the script has found it.

  • Highlight Cases: Make finding other cases with the same infringer/client easier by highlighting matches in the case list. Uses different colors for new, active, and closed/settled cases. To use: View other cases against an infringer, then click the bookmarklet. It will ask you for a name to search for and then it will highlight all cases that have that name. Bonus tip: Search for punctuation (e.g. /, #) to highlight all cases.

  • Impersonate User: Impersonate a user. To use: Either highlight a user ID number and then click the bookmarklet or click the bookmarklet and then enter a user ID number.

  • Open Case: Open a case folder from anywhere. To use: Either highlight a case number and then click on the bookmarklet, or click on the bookmarklet then enter a case number. Bonus tip: If you do the latter, you can open a specific section directly by entering a case number and a URL fragment (e.g. 11235/updates to view updates or 81321/tools to view case tools).

Miscellany

  • Compare Cases: Find image duplicates across multiple cases. To use: Open the link, add as many cases as you need. Copy/paste image names from each case into the tool, and then click compare.
